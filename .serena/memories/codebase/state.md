The mcp-zen-of-languages repository is a FastMCP server and CLI for multi-language code analysis following zen principles. It uses a modular architecture: BaseAnalyzer with template method pattern, detectors as strategy pattern, pipelines, context objects, factory, Pydantic configs, etc. The repo contains docs, tests, analyzers, rules, utilities, and has a .serena directory for local memory ignored by git. Python 3.12+ conventions are followed. Pre-commit hooks ensure quality.
